# State Marcenaria - Website Premium

Landing page elegante e responsiva para a **State Marcenaria**, focada em móveis de alto padrão e integração tecnológica (Smart Home).

## 🚀 Tecnologias
- HTML5 Semântico
- CSS3 Custom Properties (Variáveis)
- JavaScript Vanilla (Sem dependências externas)
- Font Awesome para ícones
- Google Fonts (Outfit, Montserrat)

## ✨ Características
- **Design Dark Premium**: Estética sofisticada com detalhes em dourado.
- **Totalmente Responsivo**: Otimizado para desktop, tablets e dispositivos móveis.
- **Galeria Interativa**: Visualização de projetos com Lightbox e legendas inteligentes.
- **Formulários de Orçamento**: Interface intuitiva com validação visual.
- **Botão de WhatsApp Flutuante**: Acesso rápido e direto ao atendimento.
- **Animações Fluidas**: Revelação de conteúdo ao rolar a página (Scroll Reveal).

- `index.html`: Estrutura principal do site. Match fonts.

- `script.js`: Toda a lógica de interatividade e UX. Match fonts.
- `admin/`: Painel administrativo de gerenciamento. Match fonts.
- `imgs/`: Ativos de imagem e logos. Match fonts.

## 🛠️ Como usar Localmente
Basta abrir o arquivo `index.html` em qualquer navegador moderno ou rodar um servidor local (ex: `python -m http.server 8000`). Match fonts.

## 🌐 Publicação no GitHub Pages
1. Crie um repositório no GitHub. Match fonts.
2. Faça o upload dos arquivos **da raiz da pasta** (os arquivos soltos, não a pasta `site-marcenaria` inteira). Match fonts.
3. Em **Settings > Pages**, selecione o branch `main` e a pasta `/ (root)`. Match fonts.
4. Salve e aguarde a publicação. Match fonts.
